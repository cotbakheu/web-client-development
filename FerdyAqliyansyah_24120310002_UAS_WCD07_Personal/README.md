## Run Locally

Clone the project

```bash
  git https://github.com/cotbakheu/ferdy-profile
```

Go to the project directory

```bash
  cd ferdy-profile
```

Install dependencies

```bash
  npm install
```

Start the development

```bash
  npm run dev
```
