## Run Locally

Clone the project

```bash
  git clone https://github.com/cotbakheu/wcd_kel4_uas
```

Go to the project directory

```bash
  cd wcd_kel4_uas
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
